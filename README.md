# How to start

Ruby 2.7.2
Rails 6.1.4
Database: Sqlite

--Step 1: Preparation and start the server
```
cd /path/to/project
cp .env.example .env

# Install necessary gems
bundle install

# Create and migrate database
bin/rails db:create db:migrate

# Start the server
bundle exec guard
```

--Step 2: Import this to Postman to test
